🗺️ 產品綜述：夜間安全熱點圖 (S01)

這是我們「助人 × 自利開放地圖平台」的首個 MVP 站型。S01 旨在透過群眾回報 (Crowdsourcing) 蒐集夜間不安全事件，並經過嚴格的社群驗證 (Governance) 後，在地圖上呈現為可信賴的熱點圖，協助夜歸通勤者、旅人或家長避險。

🛡️ 技術骨架與安全保證 (R3/R5)

本站型建立在 **Supabase (PostgreSQL + RLS)** 的穩固架構之上，保證數據的**安全**與**可擴展性**。

  • 資料安全 (R5 RLS): 所有提報數據均受到 **Row Level Security** 嚴格保護。只有經過審核 (Status = 'verified') 的報告，才允許公開讀取。
  • 高權限隔離 (R2): 後台的審核和信譽分數計算，全部透過 **Edge Functions** 隔離運行，絕不暴露超級金鑰 (`service_role_key`) 於前端。
  • 極致性能 (R3): 採用 **Next.js (SSG/ISR)** 架構，確保熱點圖的快速載入，同時滿足 Google Core Web Vitals (SEO) 規範。

📊 治理與數據品質 (R1)

S01 透過數據驅動的 **信譽系統**，解決群眾外包最擔心的「惡作劇」和「誤報」問題。

  1. **灰層 (Grey Status):** 所有新的報告默認為 `'grey'` 狀態，不公開顯示。
  2. **信譽權重:** 只有擁有足夠 `reputation` 分數的貢獻者提交的報告，才能更快進入審核流程。
  3. **變現反哺:** 未來將透過 B2B API 收益，反哺高信譽的貢獻者，形成正向循環。
